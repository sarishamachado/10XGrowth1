## üé¨ GLOBAL ANIMATION & INTERACTION SPECIFICATIONS

### Scroll-Triggered Animation Framework

**Library**: GSAP 3.12+ with ScrollTrigger plugin

**Global Principles**:
- Elements animate **on scroll into viewport** (trigger when element is 20% visible)
- **Stagger animations** for grouped elements (0.08-0.15s delay between items)
- **Easing functions**: easeOutQuart (general), easeOutExpo (numbers), easeOutBack (playful)
- **Duration**: 0.6-0.8s for most animations
- **Performance**: Use transform and opacity only (GPU-accelerated)

### Common Animation Patterns

**Fade + Slide Up**:
```
Initial: opacity: 0, translateY: 40px
Final: opacity: 1, translateY: 0
Duration: 0.6s
Easing: easeOutQuart
```

**Scale + Fade**:
```
Initial: opacity: 0, scale: 0.9
Final: opacity: 1, scale: 1
Duration: 0.8s
Easing: easeOutCubic
```

**Number Count-Up**:
```
Initial: 0
Final: Target number
Duration: 1.5-2s
Easing: easeOutExpo
Trigger: When fully in viewport
```

**Line/Path Drawing** (SVG):
```
stroke-dasharray: pathLength
stroke-dashoffset: pathLength ‚Üí 0
Duration: 1-1.5s
Easing: easeInOutCubic
```

### Interactive Micro-Animations

**Button Hover** (all CTAs):


### Performance Optimizations

**Intersection Observer**:
- Trigger animations only when elements are in viewport
- Disconnect after animation completes

**Throttle/Debounce**:
- Scroll events throttled to 16ms (60fps)
- Resize events debounced to 250ms

**Will-Change**:
- Apply `will-change: transform, opacity` to animated elements
- Remove after animation completes

**Reduced Motion**:
